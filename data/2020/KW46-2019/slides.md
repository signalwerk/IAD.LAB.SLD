## Input · placeholder

```html
<label for="serial-nr">Seriennummer</label>
<input
  type="text"
  id="serial-nr"
  name="Seriennummer"
  placeholder="92xx-xxxxxxxxx-xxx-x"
/>
```

## Input · Typen

```html

<input
  type="date"
  id="birthday"
  name="birthday"
  value="2000-06-12"
  min="1990-01-01"
  max="2020-12-31"
/>
```
